# Vigil 🕯️ 守夜

**凌晨睡不著的時候，打開它，點一根蠟燭，想的話留一句話。在一張顯示著地球夜面的世界地圖上，看見此刻全世界還有幾根蠟燭亮著、亮在哪些城市——幾個和你一樣醒著的陌生人。**

🔗 Live：**https://vigil-cc881.web.app**

## 如何使用

### Web 版
直接用瀏覽器開啟：[https://vigil-cc881.web.app](https://vigil-cc881.web.app)。不用註冊、不用安裝——載入的當下就已用匿名身份登入。

### iOS（加入主畫面）
1. 用 **Safari** 開啟 [守夜網頁版](https://vigil-cc881.web.app)
2. 點下方中間的「**分享**」按鈕 ↑ → 選「**加入主畫面**」
3. 點右上角「**新增**」→ 桌面出現「守夜」圖示，用法與 App 相同

## 這是什麼

夜深了，你還醒著，四周很安靜。你打開守夜，點一根蠟燭。它在一張暗色的世界地圖上、你所屬時區的某處，燃燒 30 分鐘。你可以在旁邊留下短短一句話。地圖的另一頭，還有別的蠟燭亮著——東京、倫敦、一座你沒去過的城市——有些燭光旁飄著一句話，多數只是靜靜亮著。

沒有聊天、沒有身份、沒有回覆。只有一件事：**你不是一個人醒著。**

守夜建立在一個想法上：**陪伴的最小單位，是「知道有人在」。** 一句話是上限，對話就是打擾。

### 產品規則（不可妥協）

- **一次點擊點燃蠟燭，燃燒 30 分鐘後自動熄滅。** 燃燒中不可提前熄滅、不可延長；燒完之後可以再點一根新的——新的 30 分鐘、新的一句話。守一整夜是被允許的，但每一根蠟燭都有自己完整的生滅
- **一人同時只能有一根蠟燭。** 文件 ID 就是你的匿名 uid——由結構強制保證
- **可以留一句話**（選填，上限 40 字，純文字）。這句話跟蠟燭同生共死：蠟燭熄滅即消失，不進任何歷史、不可編輯、不可回覆。永遠不解析 URL、不自動連結化
- **位置只到時區城市級**——取自瀏覽器的 `Intl` 時區（如 `Asia/Taipei`）。零 GPS、零 IP 查詢、零權限彈窗。我們不知道你在哪，只知道你的深夜屬於哪個時區
- **沒有帳號**（匿名登入，使用者無感）、**沒有名字、沒有歷史紀錄、沒有任何回覆機制**
- **燭光只會出現在夜面。** 地圖用純太陽位置數學即時畫出晨昏線；你的燭火落在地球的暗面

## 地圖

單一畫面：夜面世界地圖、燭光、一個「點燃」按鈕（旁邊一個選填的一句話輸入框），以及一行字「此刻有 N 根蠟燭亮著」。

- **可平移、縮放、環繞。** 拖曳平移（世界左右無縫相接），滾輪或雙指縮放拉近。手機打開時以你自己的時區為中心、填滿螢幕
- **點或懸停別人的燭光**，那句話與城市名會輕輕浮現——「東京 · 還有 12 分鐘」——沒留話就只有城市名與剩餘時間
- **自己的蠟燭**有細微標記與剩餘燃燒動畫；燒完後按鈕回到「再點一根」
- **安靜是刻意的。** 深夜配色、緩慢動畫、沒有音效。燭火會極輕地呼吸、點燃時淡入、燒盡時漸暗。當你的時區接近日出，地平線會極緩地透出一點暖色——夜真的會過去

同時區的蠟燭會加一個很小的隨機偏移（約 0.3°）散開，不會疊在同一個像素上；拉近後會進一步分離。

## 誠實是用基礎設施兌現的

產品的承諾由 Firestore Security Rules 強制，不靠自律：

- **蠟燭剛好燒 30 分鐘**——`expiresAt == litAt + 30min` 精確強制，不可延長
- **燃燒中的蠟燭不可覆寫**——只有前一根燒完後（`resource.data.expiresAt <= request.time`）才允許重點
- **不可刪除**（`allow delete: if false`）——你無法提前熄滅一根蠟燭，過期是唯一的出口
- **欄位級驗證**——時區必須是合法的 IANA 字串、留言必須是長度 ≤ 40 的字串、散落偏移有上界
- **讀取對所有人開放**；過期文件由 Firestore TTL 清除

以上全部由 emulator 測試套件覆蓋（`npm run test:rules`）。

## 技術

- **Vite + React + TypeScript + Tailwind CSS v4**
- **Firebase**：匿名登入（無感登入）、Firestore（`onSnapshot` 即時同步）、Hosting——完全沒有自建後端
- **地圖是手工打造、不用任何地圖庫**：內嵌 SVG 世界輪廓（來自 world-atlas 110m TopoJSON）、由 IANA `zone1970.tab` 產生的時區→座標對照表，以及自製的平移/縮放/環繞相機
- **晨昏線是純太陽位置數學**——無外部依賴

## 本地開發

```bash
npm install

# 連真實 Firebase 專案（讀 .env.local）
npm run dev

# 完整功能，連本地 emulator
npm run emulators          # Auth :9099 + Firestore :8080
npm run dev:emu

# Security Rules 測試（會自動起 Firestore emulator，需要 Java）
npm run test:rules
```

要連真實的 Firebase 專案，把 `.env.example` 複製成 `.env.local` 填入你的 web app 設定。

## 刻意不做的事

聊天、回覆、帳號、暱稱、歷史、連續打卡、通知、精確定位、音效，以及任何 engagement 優化。

蠟燭少的時候不造假——「只有你一根」也是真實。孤獨的誠實，比熱鬧的謊言重要。
